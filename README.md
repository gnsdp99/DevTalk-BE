# DevTalk - 개발자 지식 공유 커뮤니티 서비스 (2025.09.22 ~ )

## 프로젝트 소개

DevTalk은 개발자들이 편하게 질문하고 답할 수 있는 커뮤니티 서비스입니다.

## 인원

1명 (개인 프로젝트)

## 주요 기능

- 회원가입/로그인: 세션 기반 인증, CSRF 보호
- 회원탈퇴: 작성한 게시글 및 댓글 익명화
- 게시글 목록 조회: 무한 스크롤링
- 게시글 작성 / 수정 / 삭제
- 게시글 좋아요
- 댓글 목록 조회: 페이지네이션
- 댓글 작성 / 수정 / 삭제
